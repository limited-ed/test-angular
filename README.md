# Каталог фильмов
В приложении собраны популярные фильмы с высоким рейтингом на IMDB


## Запуск

### Требования

Для сборки и запуска требуется NodeJS версий  ^20.19.0 || ^22.12.0 || ^24.0.0

### Запуск браузерной версии

Установите зависимости запустив команды:
```bash
 npm install
```

Запуск приложения производится командой

```bash
start npm --prefix ./json-server run srv && ng serve
```

Откройте браузер по адресу 

   [http://localhost:4200](http://localhost:4200)
   
### Запуск версии с рендерингом на сервере

Установите зависимости запустив команды:

```bash
npm install
```

Соберите приложение при помощи команды 

```bash
ng build
```

Запуск приложения производится командой

```bash
start npm --prefix ./json-server run srv && npm run ssr
```

Откройте браузер по адресу 

   [http://localhost:4000](http://localhost:4000)

## Запуск в Docker

Приложение поддерживает возможность запуска в Docker. Для этого установите Docker по [инструкции](https://docs.docker.com/get-started/introduction/get-docker-deskto) для вашей операционной системы 

запустить приложение необходимо командой

```bash
docker compose up
```

Откройте браузер по адресу 

   [http://localhost:4000](http://localhost:4000)
   


## Тестирование

Чтобы провести unit тесты, необходимо установить зависимости запустив команду:

```bash
 npm install
```

Запустить тесты можно командой

```bash
ng test
```
или для просмотра результатов в браузере

```bash
ng test --ui
```
